## 8. 분석 방법론

### 상관관계 분석
- **피어슨 상관계수** (선형 관계)
- **스피어만 상관계수** (순위 기반, 비선형 포함)
- **최적 시차(Lag) 분석**: ±5일 범위에서 최대 상관 시차 탐색
- **단순 회귀**: R² (설명력), 기울기(방향성)

### 가중치 계산

```
가중치 = |피어슨_r| * 0.5 + R2 * 0.5  (p < 0.05 조건 충족 시)
최종 가중치 = (SP500 가중치 + KOSPI 가중치) / 2
```

### 신뢰도 점수 (0-100점)

```
데이터 충분성: 최대 30점 (행 수 기준, 300행 이상 만점)
통계적 유의성: 30점 (p < 0.05 충족 시)
상관 강도:     최대 20점 (|r| 비례)
이상값 비율:   최대 20점 (IQR 3배 기준, 이상값 0% = 20점)
```
