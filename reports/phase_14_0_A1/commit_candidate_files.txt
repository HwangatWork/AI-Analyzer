# Decision : INCLUDE in commit.
# Rule    : Repo convention tracks output/*.json artifacts.
# Evidence:
#   $ git ls-files output/ | grep -c '\.json$'
#   Multiple .json files are already tracked (e.g. output/decision.json,
#   output/final_results.json, output/ctd_bridge.json, output/ctd_weights.json,
#   output/narrative_context.json). The audit JSON follows the same pattern:

#   audit returned at any given commit, and downstream consumers (e.g.
#   dashboards, pm_quality readers) can reference it by path.

# The accompanying .gitkeep is redundant once the JSON is present but
# is intentionally kept so the directory remains visible even after a
# possible future `rm output/consensus_audit/*.json`.
