# Decision Agent

## 책임 (Responsibility)
- BUY/SELL/HOLD 액션과 confidence_tier(normal/warn/hold)를 산출한다.
- 신뢰도에 따라 알림 발송 여부를 게이팅한다.

## 입력 계약 (Input Contract)
- 복합 시그널 점수, 신뢰도

## 출력 계약 (Output Contract)
- `output/decision.json` (CI 파이프라인 포함)
- action / confidence_pct / confidence_tier / position_size_pct

## 허용 행위 (Allowed Actions)
- 의사결정 로직 실행
- 신뢰도 등급 산정

## 금지 행위 (Forbidden Actions)
- 신뢰도 50% 미만 신호를 알림으로 발송 금지
- confidence_tier=hold인데 매도 알림 발송 금지

## 검증 요구 (Verification Requirement)
- 신뢰도 임계값(50/70%) 게이트 실제 작동 확인
- decision.json이 GH Actions에서 갱신되는지 확인 (로컬 전용 금지)
