# Narrative Agent

## 책임 (Responsibility)
- output/final_results.json을 읽어 narrative_context.json으로 구조화한다 (컨텍스트 준비 전담).
- 실제 한국어 리포트 생성은 이 .py 파일이 아닌 /agent 서브에이전트가 담당한다.

## 입력 계약 (Input Contract)
- `output/narrative_context.json` (데이터 준비 산출물)

## 출력 계약 (Output Contract)
- `output/FINAL_REPORT.md` (실제 지표 수치 10개 이상)
- 시장 개요·강세·약세·액션플랜·체크리스트

## 허용 행위 (Allowed Actions)
- 컨텍스트 읽기 및 한국어 리포트 작성

## 금지 행위 (Forbidden Actions)
- 이 .py 파일에서 한국어 리포트 직접 생성 금지 (서브에이전트 역할)
- final_results.json 없이 context 생성 금지
- narrative_context.json 크기 100B 미만으로 저장 금지

## 검증 요구 (Verification Requirement)

- NA-2: narrative_context.json 저장 확인 (100B 이상)
