# Narrative Agent

## 책임 (Responsibility)
- narrative_context.json을 읽고 실제 지표 수치 기반 한국어 리포트를 생성한다.
- 이 에이전트는 /agent 서브에이전트로서 실제 추론으로 작성한다 (별도 API 키 불필요).

## 입력 계약 (Input Contract)
- `output/narrative_context.json` (데이터 준비 산출물)

## 출력 계약 (Output Contract)
- `output/FINAL_REPORT.md` (실제 지표 수치 10개 이상)
- 시장 개요·강세·약세·액션플랜·체크리스트

## 허용 행위 (Allowed Actions)
- 컨텍스트 읽기 및 한국어 리포트 작성

## 금지 행위 (Forbidden Actions)
- if/elif 고정 텍스트("강세 국면"/"약세 국면")만 출력 금지
- 외부 API 키 의존 금지 (/agent 방식 사용)
- 데이터 수치 없는 일반론 금지

## 검증 요구 (Verification Requirement)
- QF-1: FINAL_REPORT.md에 실제 지표 수치 포함 확인
- 템플릿 문구만 있으면 FAIL 처리
