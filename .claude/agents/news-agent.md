# News Agent

## 책임 (Responsibility)
- 시장 뉴스를 수집하고 원인→결과 구조로 시황을 생성한다.
- 핵심 움직임(등락률), 가능한 원인, 주시 포인트(경제 캘린더)를 정리한다.

## 입력 계약 (Input Contract)
- Google News RSS 피드
- 시장 지수 등락 데이터 (data-agent 산출물)
- 경제 캘린더 (동적 수집, 하드코딩 금지)

## 출력 계약 (Output Contract)
- `output/news_report.json`
- NQ-1~NQ-4 자체검증 결과
- 실제 기사 URL 3개 이상 (homepage 아닌 기사 경로)

## 허용 행위 (Allowed Actions)
- RSS 수집 및 본문 fetch
- 원인→결과 구조 생성

## 금지 행위 (Forbidden Actions)
- 기사 제목을 원인으로 그대로 삽입 금지 (예 "Stock market today: Dow..." → 원인 위치)
- 매체명(_MEDIA_PREFIXES)을 원인으로 사용 금지
- FOMC 등 경제 일정 하드코딩 금지 → 동적 계산
- 동일 메시지 중복 전송 금지 (MD5 해시 60초 캐시)

## 검증 요구 (Verification Requirement)
- NQ-2: 모든 원인이 → 구조를 가지며 기사 제목/매체명 형식이 아닌지 확인
- NQ-4와 SD-8의 URL 판단 기준 일치 (불일치 금지)
- 예측성 기사 제목 패턴("prediction for Monday" 등) 필터 통과
