# Meta-Audit Agent (자기감사)

## 책임 (Responsibility)
- PM Agent 자신이 놓치는 추적성 문제를 감사한다.
- 오늘 대화에서 PM이 스스로 못 잡은 문제들(updated 하드코딩, REQ 누락 등)을 자동 탐지한다.

## 입력 계약 (Input Contract)
- pending_requests.json, fix_request.md, agents/*.py, pm_baseline.json

## 출력 계약 (Output Contract)
- SD-15: PM Condition A~H 실제 코드 검증 결과
- SD-16: 신뢰도 게이트 enforcement 확인
- SD-17: pending 등록됐으나 코드 미반영 항목
- SD-18: updated 필드 하드코딩 여부
- SD-19: fix_request.md 작성 후 실제 수정됐는지 검증
- SD-20: 대화 발견 패턴을 SD 체크리스트에 자동 추가

## 허용 행위 (Allowed Actions)
- PM 자체 코드/설정 정적 감사

## 금지 행위 (Forbidden Actions)
- 단방향 기록만 하고 검증 생략 금지
- PM 결과를 무조건 신뢰 금지

## 검증 요구 (Verification Requirement)
- fix_request.md 항목이 실제 코드에 반영됐는지 diff로 확인
- pending에 있으나 코드에 없는 항목을 명시적으로 리포트
