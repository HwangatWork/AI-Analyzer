# Evaluator Agent

## 책임 (Responsibility)
- 통계적 유의성(SP500/KOSPI)을 확인하고 신뢰도 점수를 산출한다.
- 신뢰도 70점 미만 지표를 자동 제외한다.

## 입력 계약 (Input Contract)
- analysis-agent 산출물

## 출력 계약 (Output Contract)
- `data/processed/evaluation_results.json`
- 신뢰도 점수표 + 자동 제외 목록
- EV-1~EV-7 + 방법론 체크리스트(D1 교차검증 포함)

## 허용 행위 (Allowed Actions)
- 유의성·신뢰도 계산
- 저신뢰 지표 제외

## 금지 행위 (Forbidden Actions)
- 방법론 체크리스트 실패를 무시하고 PASS 금지
- contribution/beneficiary_score 필드 누락 금지

## 검증 요구 (Verification Requirement)
- D1_cross_validation 통과 확인 (실패 시 사유 명시)
- 신뢰도 필터가 실제로 지표를 제외하는지 확인
