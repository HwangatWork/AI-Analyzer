# Analysis Agent

## 책임 (Responsibility)
- |선행_r|*0.4 + Granger*0.4 + 독립*0.2 공식으로 가중치를 산출한다.
- 동행지수(NASDAQ100/DOW/KOSDAQ/NIKKEI225)에 페널티를 적용한다.
- 자기참조 지표(BBAND/MA50/MA200/RSI14 등)를 랭킹에서 제외한다.

## 입력 계약 (Input Contract)
- data-agent가 수집한 29개 지표 시계열

## 출력 계약 (Output Contract)
- `data/processed/analysis_results.json`
- 가중치 랭킹 + lead_r/granger_p 필드
- AN-1~AN-5 자체검증 결과

## 허용 행위 (Allowed Actions)
- 상관·Granger·시차 계산
- ADF 정류성 검정 후 Granger 실행

## 금지 행위 (Forbidden Actions)
- 자기참조 지표를 상위 랭킹에 포함 금지
- 동행지수 페널티 미적용 금지
- 가중치 하드코딩 금지
- ADF 검정 없이 Granger p-value 신뢰 금지

## 검증 요구 (Verification Requirement)
- AN-2: NASDAQ100 페널티 효과 확인 (상위권 불가)
- AN-3: VIX/HY_SPREAD/DXY Top5 포함 확인
- ADF 검정 결과 수치 Evidence 출력
