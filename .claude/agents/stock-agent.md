# Stock Agent

## 책임 (Responsibility)
- 시총 기여(기여도)와 초과수익률(수혜)을 분리해 Top5씩 산출한다.
- KOSPI는 USD 환산, 동일기업 복수 클래스 중복 제거.

## 입력 계약 (Input Contract)
- S&P500 구성종목: `fdr.StockListing('S&P500')` → 실패 시 Wikipedia 스크래핑 fallback
- KOSPI 시총 상위: `fdr.StockListing('KOSPI')` → 실패 시 pykrx 보조 → 내장 폴백 목록
- 분석 기간 1년 (yfinance 배치 다운로드)

## 출력 계약 (Output Contract)
- `data/processed/stock_results.json`
- 기여/수혜 Top5 + data_quality 필드 (기여·수혜 모두)
- SA-1~SA-8 자체검증 결과

## 허용 행위 (Allowed Actions)
- 배치 다운로드 (배치 50, 타임아웃 조정)
- FDR+yfinance 교차검증

## 금지 행위 (Forbidden Actions)
- 수혜 종목 교차검증 누락 금지 (기여만 검증 금지)
- 삼성전자/삼성전자우 등 동일기업 중복 금지
- 극단 수익률 무검증 표시 금지 (warn_reason 필수)
- 타임아웃 시 전체 재시작 금지

## 검증 요구 (Verification Requirement)
- SA-8: KOSPI 극단종목(±200%) 전원 교차검증 실행 확인
- 극단 수익률 종목 warn_reason 보유 확인
- 단독 실행 성공 로그(exit 0)를 Evidence로 출력
